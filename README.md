# 🩺 메디프라이스 (MediPrice) - 로컬 프로젝트 가이드

이 폴더는 `mediprice.pages.dev` 사이트의 로컬 관리 및 고도화를 위해 구축된 작업 공간입니다.

## 📂 폴더 구조 및 구성 파일

*   `index.html`: 다크 모드 토글과 예상 의료비 계산기가 탑재된 웹 페이지 메인 소스코드
*   `style.css`: 유리모피즘 디자인과 다크 모드 테마, 가격 분포 진행 바(Progress Bar) 스타일이 적용된 스타일시트
*   `app.js`: dynamic dropdown 필터링(data.json 기반), 정렬, 다크 모드 상태 저장 및 의료비 연산 프론트엔드 로직
*   `data.json`: 기존 데이터 대비 전국 17개 시도(서울, 경기, 부산, 대구, 인천 등)로 대폭 확충된 575개의 실거래 가격 데이터
*   `generate_data.py`: Gemini 2.5 Flash API와 하이브리드 알고리즘을 이용해 데이터를 추가 및 갱신해주는 파이썬 자동화 스크립트
*   `.env`: 데이터 생성용 Gemini API Key 설정 파일

---

## ⚡ 1. 데이터 업데이트 & 자동화 (언제든 추가 가능)

언제든지 더 많은 데이터를 추가하거나 가격을 초기화하고 싶다면 다음 명령어를 터미널에서 실행하십시오:
```bash
python generate_data.py
```
*이 스크립트는 실행 시 자동으로 `.env`의 API 키를 연동해 전국 단위의 현실적인 병원 데이터를 생성하고 `data.json`을 갱신합니다.*

## 🚀 2. GitHub를 통한 Cloudflare Pages 업데이트 및 배포 방법

이미 Cloudflare Pages가 특정 GitHub 저장소(Repository)에 연결되어 있으므로, 이 폴더의 변경된 파일들을 해당 저장소에 푸시(Push)하면 자동으로 배포됩니다.

### 방법 A: VS Code(비주얼 스튜디오 코드)를 사용하는 경우 (가장 간편)
1. **VS Code**로 이 `mediprice` 폴더를 엽니다.
2. 좌측 메뉴에서 **소스 제어(Source Control)** 탭(단축키: `Ctrl + Shift + G`)을 선택합니다.
3. **[원격 저장소 구성]** 또는 **[리포지토리 초기화]**를 누른 뒤 본인의 GitHub 저장소 URL을 연결합니다.
4. 변경 사항을 스테이징(`+` 버튼)하고 커밋 메시지(예: `feat: UI 고도화 및 데이터 확충`)를 입력한 후 **[커밋]** 및 **[푸시]**를 누릅니다.

### 방법 B: Git CLI 터미널을 사용하는 경우 (Git이 설치된 환경)
이 폴더 경로에서 터미널을 열고 아래 명령어를 순서대로 실행하십시오:
```bash
# 최초 1회만 실행 (기존 저장소가 없을 경우 초기화 및 연결)
git init
git remote add origin <본인의_깃허브_레포지토리_주소.git>

# 파일 업로드 및 푸시
git add .
git commit -m "feat: UI 고도화 및 데이터 확충"
git branch -M main
git push -u origin main --force
```

### 방법 C: Git이 설치되어 있지 않은 경우 (파이썬 업로드 스크립트 사용)
컴퓨터에 Git이 설치되어 있지 않아도, 아래 파이썬 스크립트를 실행하여 깃허브 API를 통해 직접 업로드할 수 있습니다:
```bash
python push_to_github.py
```
*스크립트 실행 시 깃허브 저장소 이름(예: `계정명/저장소명`)과 개인 액세스 토큰(PAT)을 입력하면 파일을 자동으로 업로드 및 커밋해 줍니다. 입력 정보는 `git_config.json`에 저장되어 다음 실행 시부터 생략 가능합니다.*

## 🔍 3. 로컬에서 미리보기 테스트

배포하기 전에 웹 브라우저에서 잘 작동하는지 보고 싶다면 아래 명령어를 입력하여 로컬 서버를 띄울 수 있습니다:
```bash
python -m http.server 8000
```
그 후 크롬 등 브라우저를 열고 `http://localhost:8000` 주소로 접속하면 다크 모드와 계산기, 전국 데이터를 직접 테스트해볼 수 있습니다.
